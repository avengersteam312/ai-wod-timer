"""EMOM = Every Minute On the Minute.\n\nCONCEPT:\n- Athlete performs prescribed work at the start of each interval\n- Remaining time in the interval is rest before next round begins\n- Each interval is SEPARATE (not one combined interval)\n\nINTERVAL DURATION BY TYPE:\n- EMOM = 60 seconds per interval\n- E2MOM (Every 2 Minutes) = 120 seconds per interval\n- E3MOM (Every 3 Minutes) = 180 seconds per interval\n- E[N]MOM = N * 60 seconds per interval\n\nINTERVAL STRUCTURE:\n- Create one interval per round/minute\n- type: \"work\" for each round\n- duration: based on EMOM variant (60s, 120s, 180s, etc.)\n\nEXAMPLES:\n\nStandard EMOM:\n\"EMOM 10: 5 Pull-ups, 10 Push-ups\"\n→ intervals: [{duration: 60, type: \"work\"}, {duration: 60, type: \"work\"}, ... x10]\n→ movements: [{name: \"Pull-ups\", reps: 5}, {name: \"Push-ups\", reps: 10}]\n\nE2MOM (every 2 minutes):\n\"E2MOM 12 min: 3 Power Cleans 155#, 6 Bar Facing Burpees\"\n→ intervals: [{duration: 120, type: \"work\"}, {duration: 120, type: \"work\"}, ... x6]\n→ movements: [{name: \"Power Cleans\", reps: 3, weight: \"155#\"}, {name: \"Bar Facing Burpees\", reps: 6}]\n\nE3MOM (every 3 minutes):\n\"E3MOM x 15: 400m Run\"\n→ intervals: [{duration: 180, type: \"work\"}, {duration: 180, type: \"work\"}, ... x5]\n→ movements: [{name: \"Run\", distance: \"400m\"}]\n\nAlternating EMOM:\n\"EMOM 12 - Odd: 15 Cal Row, Even: 12 KB Swings\"\n→ intervals: [{duration: 60, type: \"work\"}, {duration: 60, type: \"work\"}, ... x12]\n→ movements: [{name: \"Cal Row\", calories: 15}, {name: \"KB Swings\", reps: 12}]\n\nEMOM with explicit rest:\n\"EMOM x5:00, rest 1:00, EMOM x5:00\"\n→ intervals: [{duration: 60, type: \"work\"} x5, {duration: 60, type: \"rest\"}, {duration: 60, type: \"work\"} x5]\n\nEvery 90 seconds:\n\"Every 90 sec for 9 min: 5 Deadlifts 225#\"\n→ intervals: [{duration: 90, type: \"work\"}, {duration: 90, type: \"work\"}, ... x6]\n→ movements: [{name: \"Deadlifts\", reps: 5, weight: \"225#\"}]"""
