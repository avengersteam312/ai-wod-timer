"""INTERVALS = Custom work/rest timing (NOT Tabata 20/10).\n\nCONCEPT:\n- Structured work and rest periods with custom durations\n- Unlike Tabata (fixed 20/10), intervals can have any work/rest ratio\n- Common in HIIT training, circuit workouts, and conditioning\n\nINTERVAL STRUCTURE:\n- Alternating work and rest intervals\n- Create separate interval for each work and rest period\n- type: \"work\" for active periods, \"rest\" for recovery periods\n\nEXAMPLES:\n\nBasic work/rest:\n\"30/15 x 4 rounds\"\n→ intervals: [{duration: 30, type: \"work\"}, {duration: 15, type: \"rest\"}] x4 = 8 intervals\n\nOn/off format:\n\"40 on 20 off x 6: Burpees\"\n→ intervals: [{duration: 40, type: \"work\"}, {duration: 20, type: \"rest\"}] x6 = 12 intervals\n→ movements: [{name: \"Burpees\"}]\n\nSeconds notation:\n\"45 seconds work / 15 seconds rest x 8\"\n→ intervals: [{duration: 45, type: \"work\"}, {duration: 15, type: \"rest\"}] x8 = 16 intervals\n\nCircuit with intervals:\n\"30/15: Squats, Push-ups, Lunges x 3 rounds\"\n→ intervals: [{duration: 30, type: \"work\"}, {duration: 15, type: \"rest\"}] x9 = 18 intervals\n→ movements: [{name: \"Squats\"}, {name: \"Push-ups\"}, {name: \"Lunges\"}]\n\nLong work periods:\n\"2 min work / 1 min rest x 5: 400m Run\"\n→ intervals: [{duration: 120, type: \"work\"}, {duration: 60, type: \"rest\"}] x5 = 10 intervals\n→ movements: [{name: \"Run\", distance: \"400m\"}]\n\nUnequal work/rest:\n\"60/30 x 4: KB Swings, Goblet Squats\"\n→ intervals: [{duration: 60, type: \"work\"}, {duration: 30, type: \"rest\"}] x4 = 8 intervals\n→ movements: [{name: \"KB Swings\"}, {name: \"Goblet Squats\"}]"""
