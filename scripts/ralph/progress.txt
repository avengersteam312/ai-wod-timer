# Ralph Progress Log
Started: Sun Feb  1 13:07:03 CST 2026

## Codebase Patterns
- Frontend uses Vue 3 + TypeScript + Vite in `frontend/` directory
- TypeScript checking: `npx vue-tsc --noEmit` (or `npm run build` which includes vue-tsc)
- Working directory for frontend commands: `frontend/`
- Capacitor packages installed: @capacitor/core, @capacitor/cli, @capacitor/ios, @capacitor/android (v8.0.2)
- Capacitor config: `frontend/capacitor.config.ts` with appId 'com.wodtimer.app', webDir 'dist'
- Native storage: Use `capacitorStorage` from `src/services/capacitorStorage.ts` instead of localStorage directly
- Haptic feedback: Use `useHaptics` composable from `src/composables/useHaptics.ts` for native haptic feedback
- Keep awake: Use `useKeepAwake` composable from `src/composables/useKeepAwake.ts` for screen wake lock
- Platform detection: Use `src/utils/platform.ts` for isNative(), isIOS(), isAndroid(), isWeb(), getPlatform()
- Safe area utilities: Use global CSS classes `safe-area-pt`, `safe-area-pb`, `safe-area-pl`, `safe-area-pr`, `safe-area-inset` from `src/style.css`
- Headers in views use `safe-area-pt` class, BottomNav uses `safe-area-pb` and horizontal safe areas
- Android back button: App.vue handles backButton event from @capacitor/app plugin with exit confirmation dialog
